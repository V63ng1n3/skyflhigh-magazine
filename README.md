# 🦅 SkyFlHigh Magazine

**Rise. Inspire. FlHigh.**

The official SkyFlHigh Magazine — a responsive web magazine that introduces the SkyFlHigh movement: the culture, philosophy, vocabulary, merchandise, and access to the Beginner's Guide.

## 📄 Files

| File | Purpose |
|------|---------|
| `index.html` | **Main magazine** — responsive web magazine (open this in any browser) |
| `book.html` | Boxing manual source (compiled into PDF via Puppeteer) |
| `compile.js` | Compiles `book.html` → `skyflhigh_boxing_manual.pdf` |
| `animation_preview.html` | Interactive SVG animation — Step In / Jab / Reset sequence |
| `Image folder/` | Magazine imagery and scanned handwritten notes |

## 🚀 How to View the Magazine

Just open `index.html` in any browser. Works on mobile, tablet, and desktop.

## 📥 How to Compile the Boxing Manual PDF

```bash
npm install
npm run compile
```

Output: `skyflhigh_boxing_manual.pdf`

The PDF is linked from the magazine's download section.
